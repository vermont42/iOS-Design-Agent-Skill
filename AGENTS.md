# Agent Guidelines

This repository contains the **iOS Design Agent Skill** — a design-critique skill for iOS/SwiftUI apps.

## In Scope

- iOS/SwiftUI UI design review and critique
- Typography, color systems, spatial composition, motion, and atmospheric depth
- SwiftUI modifier suggestions and code examples
- Accessibility (Dynamic Type, dark mode, VoiceOver, reduced motion)
- Apple Human Interface Guidelines alignment

## Out of Scope

- General Swift language guidance (use a Swift skill instead)
- Architecture patterns (MVVM, VIPER, TCA, etc.)
- Backend, networking, or data layer code
- Non-Apple platforms
- Build system, CI/CD, or tooling configuration
